Welcome to Heart Track

A low-cost IoT-enabled app for heart rate and blood oxygen monitoring.

Heart Track is a project aimed at providing real-time monitoring of heart rate and blood oxygen levels, offering easy access to crucial health data.

Project Info

Heart Track is a low-cost, IoT-enabled web app for monitoring heart rate and blood oxygen levels. Users are reminded to take periodic measurements, which are displayed on a responsive web application. Measurement time and frequency are customizable by users or physicians.

Web Application:

Developed with HTML, CSS, and JavaScript, the app enables users to create accounts, manage devices, and view data. Node.js, Express, and MongoDB power the backend, while the frontend is responsive across desktop, tablet, and mobile devices. Users can view weekly summaries and detailed daily data on heart rate and oxygen levels.

Hardware:

The project utilizes the Argon IoT Starter Kit from Particle, featuring the Argon microcontroller with Wi-Fi and Bluetooth. It also includes the MAX30102 sensor for monitoring heart rate and oxygen levels, along with essential accessories like jumper wires and a breadboard.
